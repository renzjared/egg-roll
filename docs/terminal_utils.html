terminal_utils.load_localization(language_code: str | None = None) → dict[str, str | list[str]]

Loads the localization file based on the provided language code.

If no language code is provided, the function will read the set language code from the “settings.json” file located in the “localization” directory. The default language is “en” (English).

Parameters:

language_code (str | None) – The language code for the localization file. If None, the set language code will be used.

Returns:

A dictionary containing the localization data.

If the localization file structure is invalid, an empty dictionary is returned.

Return type:

EggRollLocalization
