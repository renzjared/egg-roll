class leaderboard_utils.Leaderboard(level_name: str)

Bases: object

A class to manage and display game leaderboards.

filename

A JSON file where all game leaderboards are stored.

Type:

str

level_name

The name of the game level associated with the leaderboard.

Type:

str

update(player_name: str, score: int) → None

Updates the leaderboard with a new score.

The leaderboard is sorted to only keep the top 10 scores.

Parameters:

  • player_name (str) – The name of the player to be added.

  • score (int) – The player’s score.

display() → None

Displays the leaderboard for the current game level.

If no leaderboard exists for the current level, an error message is shown.

leaderboard_utils.read_leaderboards() → dict[str, list[dict[str, str | int]]]

Reads the leaderboard from the JSON file.

If the file exists but does not have a valid structure, an empty dictionary is returned.

Returns:

A dictionary containing leaderboard data for all game levels, or an empty dictionary if the file does not exist or is invalid.

Return type:

EggRollLeaderboards
